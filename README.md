## Study App

Учебное консольное приложение на Python для работы с карточками.

## 🚀 Функционал
* Просмотр и управление карточками через консольный интерфейс.
* Изучение материалов в удобном текстовом формате.

## 🛠 Требования и запуск

Для работы приложения необходим **Python 3.x**.

1. Склонируйте репозиторий или скачайте файл `study_app.py`.
2. Запустите скрипт через терминал:
```bash
   python study_app.py
